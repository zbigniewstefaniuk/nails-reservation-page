<h1 align="center">Setup Instructions</h1>

1. Clone the repo

```sh
git clone https://github.com/zbigniewstefaniuk/todo-react-app
```

2. Install NPM packages

```sh
npm i
```

3. Start app

```sh
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
